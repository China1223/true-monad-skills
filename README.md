# true-monad-skills

TRAE Skills collection for multi-version project coordination and management.

## Skills

| Skill | Description |
| --- | --- |
| [multi-version-coordination](multi-version-coordination/) | Multi-version bug fix coordination: git commit, restart, and cross-version update sync |

## Usage

Clone this repo and copy the desired skill directory into your project's `.trae/skills/` folder:

```bash
git clone git@github.com:China1223/true-monad-skills.git
cp -r true-monad-skills/multi-version-coordination your-project/.trae/skills/
```
